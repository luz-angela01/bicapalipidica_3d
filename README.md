## Bicapa Lipídica 3D — Blender + Geometry Nodes

Este proyecto es una visualización 3D de una **membrana celular** basada en el
modelo del **mosaico fluido**, creada en *Blender* mediante **Geometry Nodes**.

### Características
- Bicapa lipídica generada de forma procedural
- Proteínas integrales estilizadas con colores pastel biomédicos
- Iluminación con HDRI
- Render final en Cycles (700 samples)

### Inspiración
Tutorial de YouTube:  
“Making (Procedural) Membranes | Blender for Biochemists | Geometry Nodes” — Brady Johnston.

### Objetivo
Proyecto de práctica para mi portafolio en **visualización científica y animación biomédica**.
